# 逆波兰表达式 (RPN) 详解

逆波兰表达式（也叫**后缀表达式**）是一种不需要括号的数学表达式书写方式。

## 1. 什么是逆波兰表达式？
普通的算术表达式（中缀表达式）是 `1 + 2`，而后缀表达式则是 `1 2 +`。

### 核心规则：
- 操作数在前，运算符在后。
- 遇到数字就入栈。
- 遇到运算符就弹出栈顶的两个数字进行计算。
